In this course, we will use Earth Engine, and you will need to develop skills in both remote sensing and programming. The language of this course is JavaScript.  

Objective: Get setup with GEE and learn how to submit future labs.

  1. Step 1 - Sign up for an Earth Engine account
  2. Go to https://earthengine.google.com/
  3. You can visit the website and get familiar with the information Google presents about Earth Engine. 
  4. Click the Sign-Up page and Fill out the form.  https://signup.earthengine.google.com/
  5. Normally signup is very quick after request.
  6. Step 2 - Access the Earth Engine Code Editor
  7. Once your account is opened and approved by Google, go to https://code.earthengine.google.com/
  8. The Code Editor is an integrated Earth Engine JavaScript API development environment. It offers an easy way to type, debug, run, and manage code.
  10. Part 3 - Coding
  11. When first learning a new language and getting used to a new programming environment, it is customary to make a program to display the words “Hello World.” This is a fun way to start coding that shows you how to give input to the program and how to execute it. It also shows where the program displays the output. Doing this in JavaScript is quite simple.
  12. Type the following code into line one of your script.
  13. print('Hello World');
  14. The line of code above uses the JavaScript print function to print the text “Hello World” to the screen. Once you enter the code, click the Run button. The output will be displayed on the upper right-hand panel under the Console tab.
  16. Part 4 - Save your Code
  17. You just wrote your first Earth Engine script and may want to save it. Click the Save button.
  19. If this is your first time using the Code Editor, you will be prompted to create a home folder. This is a folder in the cloud where all your code will be saved. You can pick a name of your choice, but remember that it cannot be changed and will forever be associated with your account. A good choice for the name would be your Google Account username.
  20. Once your home folder is created, you will be prompted to enter a new repository. A repository can help you organize and share code. Your account can have multiple repositories and each repository can have multiple scripts inside it. To get started, you can create a repository named “default”
  21. Finally, you can save your script inside the newly created repository. Enter the name “hello_world” and click OK

  23. Once the script is saved, it will appear in the script manager panel. The scripts are saved in the cloud and will always be available to you when you open the Code Editor.

  24. Part 5 - Share your Code
  25. Click on the Get link button. This will generate a URL for your code.
